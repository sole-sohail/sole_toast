# Sole Toast

A gooey, morphing toast for Flutter — organic pill → blob animation,
light/dark/glossy modes, semantic types, and iPhone Dynamic Island
choreography. Zero dependencies.

> Full documentation is completed alongside v0.1.0 — see the example app.

## Quick start

```dart
MaterialApp(
  builder: SoleToast.init(),
  home: ...,
);

SoleToast.success('Saved', description: 'Your changes have been synced.');
```
