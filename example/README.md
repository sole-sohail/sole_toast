# sole_toast example

A small playground for every Sole Toast feature.

```bash
flutter create .   # one-time: generates the ios/ and android/ folders
flutter run
```

Try the mode selector (light / dark / glossy), fire each toast type, and
enable **Island preview** to see the Dynamic Island choreography on any
device or simulator. On iPhones with a real Dynamic Island it activates
automatically — a welcome toast plays on launch.
